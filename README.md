## 一、项目目标
订票系统包括浏览当前所有火车的信息，包含起始站与终点站，途径的站点，发车日期和时间，到站时刻、余票情况、座位和票价；查询功能包含对车次的查询，也可以按照区间进行查询，其次是对已经购买订单的查询，线下也可以进行购票，数据库中的余票情况相应减少，购买功能包含选定日期、车次和区间后进行购买，同时线下购票时也会相应减少数据库中的车票记录，目的是方便人们日常购买车票。
